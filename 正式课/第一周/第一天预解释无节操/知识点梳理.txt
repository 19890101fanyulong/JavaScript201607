git 安装上  
github.com  用户名和密码  
git clone 地址  
git pull origin master  更新  .git

git init把当前文件夹初始化成一个仓库
git remote add origin 地址  让本地仓库建立和远程仓库的连接
git pull origin master 

git add -A  把添加的文件添加到暂存区 
git commit -m'本次提交的描述'  版本号
git push origin master 提交到github远程仓库

.ignore忽略上传的文件放在这里
